Base64

Base64 is a binary-to-text encoding scheme. When encoding binary with Base64, three bytes (8-bits) at a time are interpreted as four 6-bit digits. 2 to the power of 6 equals 64, hence the name. These digits are then turned into ASCII characters using the Base64 table.

The table uses all alphanumeric characters from ASCII and +, / and = as a padding character. This padding character is used when the number of bytes of the input binary is not divisible by 3.
